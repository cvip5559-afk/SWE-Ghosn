# Ghosn Platform 🌱

## Overview
Ghosn is a web-based platform that enables residents to report areas that need tree planting and allows volunteers to participate in improving urban green spaces.

## Features
- Submit reports for locations lacking greenery  
- View, edit, and delete reports  
- Search and filter reports by severity level (1–5)  
- Register and log in as Resident or Volunteer  
- Volunteer participation in reported locations
